## 目标用户

**第一优先**：个人上班族

- 工作场景碎片化
- 缺乏AI使用习惯
- 需要开箱即用

**第二优先**：企业AI转型

- 不知道怎么开始
- 数据安全顾虑
- 需要多人协作

***

## 「爱马仕能力」设计

精品化体验 = 爱马仕级品质

### 精品场景包

| 场景包      | 核心能力           |
| -------- | -------------- |
| **会议助手** | 会议纪要、待办提取、任务分配 |
| **邮件管家** | 智能回复、分类、批量处理   |
| **文档工厂** | 周报/月报/方案一键生成   |
| **审批加速** | 报销单校验、流程催办     |
| **日程优化** | 智能排期、时间推荐      |

### 精品交互

- 渐进式引导：AI主动问「今天帮你做什么？」
- 极简操作：一句话需求，自动执行
- 结果可追溯：每步操作透明，用户可修改

### 企业级能力

- 数据隔离：企业数据不出域
- 权限矩阵：角色差异化
- 审计日志：操作可追溯
- 多租户：支持多企业

***

## 实施路径

### Phase 1：MVP（6-8周）

- Web端 + 飞书/钉钉插件
- 3个核心场景包
- 单用户版本

### Phase 2：企业版（8-12周）

- 团队管理、权限控制
- 企业知识库集成
- 数据隔离、多人协作

### Phase 3：生态扩展

- 开放API/插件市场
- 行业解决方案
- 私有化部署


## 竞争策略

**不要做**：

- 不要和ArkClaw比云端部署
- 不要和OpenClaw比技术深度

**要做**：

- 聚焦「场景深度」和「企业级体验」
- 「精品化」是核心差异点
- 先小验证，再押大注

***

## 下一步行动

1. **验证场景**：找10个上班族访谈工作痛点
2. **做MVP**：选最高频场景（建议会议纪要），做到极致
3. **小规模投放**：验证用户是否愿意用、愿意付费
4. **数据迭代**：用户行为告诉你方向，不凭直觉
