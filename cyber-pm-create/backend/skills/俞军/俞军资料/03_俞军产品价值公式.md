# 俞军产品价值公式

## 核心公式

```
用户价值 = 新体验 - 旧体验 - 替换成本
```

## 公式解读

### 组成要素

1. **新体验**
   - 用户使用新产品/新功能获得的体验
   - 包含：功能价值、情感价值、社交价值等

2. **旧体验**
   - 用户使用原有产品的体验
   - 可能是竞品，也可能是"不用"的体验

3. **替换成本**
   - 用户从旧产品切换到新产品的成本
   - 包含：学习成本、数据迁移成本、习惯改变成本、社会关系成本等

## 应用场景

### 产品决策
- 判断一个功能是否值得上线
- 评估竞品威胁程度
- 制定产品优先级

### 公式变形

当替换成本较高时，需要：
- 大幅提升新体验
- 或者降低用户的替换成本（如补贴、优惠）

### 经典案例

**滴滴 vs 出租车**
- 新体验：更便捷的叫车体验、更透明的价格
- 旧体验：路边等车、不确定是否有车
- 替换成本：下载APP、绑定支付、改变出行习惯

**微信红包 vs 传统红包**
- 新体验：社交+金钱的创新玩法
- 旧体验：纸质红包
- 替换成本：低（操作简单有趣）

## 实践要点

1. **量化评估**：尝试将各要素量化打分
2. **用户视角**：从目标用户角度评估，而非工程师视角
3. **动态眼光**：替换成本会随用户习惯改变而降低
