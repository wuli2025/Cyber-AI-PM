## 阅读建议

### 入门路线（新人）

```
第1步：《人人都是产品经理》- 了解产品经理是什么
     ↓
第2步：《点石成金》- 了解可用性设计
     ↓
第3步：《启示录》- 了解产品价值
```

```

第2步：《用户体验要素》- 系统性框架
     ↓
第3步：俞军产品方法论 - 本土实践
```
