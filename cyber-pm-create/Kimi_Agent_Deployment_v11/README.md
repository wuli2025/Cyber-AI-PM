# Cyber AI PM - 顶级产品思维聚合站

> 聚合全球顶级产品思维，让创意专业化落地

## 🎯 项目简介

AI 产品思维实验室，聚合六位全球顶尖产品智囊：

- 🍎 **乔布斯** - 交互与美学偏执者
- 🚀 **马斯克** - 第一性原理践行者  
- 💬 **马化腾** - 社交产品之王
- ⚡ **张一鸣** - 推荐算法极致者
- 📝 **俞军** - 中国产品经理教父
- 🎓 **教员** - 战略思维大师

## ✨ 核心特性

- **多角色切换**：一键切换不同产品思维视角
- **专业 PRD 生成**：从创意到可落地文档
- **多元思维碰撞**：六大维度全方位审视创意
- **流式响应**：实时获取 AI 回复

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动服务
npm start
```

然后访问 http://localhost:8080

## 💡 创作理念

> 迈入人工智能新时代，技术工具已然成熟可落地，当下真正稀缺的是独到创新创意、深度行业认知与精准用户洞察。

本项目以 AI 为载体，承接文案编撰、格式梳理等机械繁杂的重复性工作，让创意回归本源。聚合六位顶尖产品智囊，从商业价值、用户心智、技术实现、交互体验、工程落地、版本迭代六大维度，全方位拆解审视创意需求。
